# Code of Conduct / 行为准则

## Our Standards / 我们的标准

We want a collaborative, respectful, and technically focused community.

我们希望社区保持协作、尊重，并聚焦技术本身。

Examples of expected behavior:

- constructive technical feedback
- respectful language
- empathy toward contributors with different experience levels
- focus on project goals and user impact

鼓励的行为包括：

- 建设性的技术反馈
- 尊重他人的表达方式
- 对不同经验层级的贡献者保持耐心
- 聚焦项目目标与用户价值

Examples of unacceptable behavior:

- harassment or personal attacks
- hate speech or discriminatory language
- doxxing or sharing private information without permission
- intentional disruption of discussions or reviews

不可接受的行为包括：

- 骚扰或人身攻击
- 仇恨或歧视性言论
- 未经允许公开他人隐私信息
- 故意破坏讨论或评审流程

## Enforcement / 执行方式

Project maintainers may warn, temporarily ban, or permanently ban participants who violate this code of conduct.

维护者有权对违反准则的参与者进行警告、临时封禁或永久封禁。

## Reporting / 举报方式

If you observe abusive behavior, report it privately to project maintainers and include links, logs, and timestamps when possible.

如果你观察到不当行为，请私下联系维护者，并尽量附上链接、日志和时间戳。
